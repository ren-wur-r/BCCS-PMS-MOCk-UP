# 資安健診專案工作板 Milestone List

本文件定義「資安健診專案」在專案管理系統中可使用的標準 Milestone 清單。
每一個 Milestone 對應一個明確的專案階段，並具備可驗證的完成條件與必要產出，用於專案進度追蹤與結案判定。
本清單不依賴報告書製作完成與否。

---

## Milestone 0：專案已立案（Project Initiated）

### 目的

確認資安健診專案已正式成立，並可進入專案管理流程。

### 完成條件（Definition of Done）

- 專案已建立並具備唯一識別
- 專案負責人與客戶窗口已確認

### 必要產出

- 專案基本資料（專案名稱、客戶、期間、專案類型）
- 專案負責人與聯絡窗口清單

---

## Milestone 1：專案啟始完成（Project Kickoff Completed）

### 對應流程階段

專案啟始與定義階段

### 完成條件

- 專案目標、範圍與驗收邊界已確認
- 啟始會議已完成

### 必要產出

- 專案工作計畫書
- 啟始會議紀錄或簡報

---

## Milestone 2：檢測規劃完成（Assessment Planning Completed）

### 對應流程階段

資料準備與檢測規劃階段

### 完成條件

- 受測系統與範圍已完整列出
- 檢測項目與方法已定義
- 檢測時程與資源已排定

### 必要產出

- 受測系統與環境清單
- 檢測項目與方法對照表
- 檢測計畫文件

---

## Milestone 3：現場檢測完成（Field Assessment Completed）

### 對應流程階段

現場執行與資料蒐集階段

### 完成條件

- 所有計畫內檢測項目皆已執行完成
- 原始檢測資料已完整保存

### 必要產出

- 掃描結果、設定檔、日誌、截圖等原始資料
- 檢測行動紀錄
- 訪談或現場確認紀錄

---

## Milestone 4：初步分析完成（Initial Analysis Completed）

### 對應流程階段

初步分析與風險分級階段

### 完成條件

- 所有檢測資料已完成整理與分析
- 弱點與風險已完成分級

### 必要產出

- 弱點與問題清單
- 風險等級評估表
- 問題分類與統計資料
- 初步改善建議摘要

---

## Milestone 5：結果確認完成（Findings Validated）

### 對應流程階段

結果確認與收斂階段

### 完成條件

- 分析結果已與客戶或內部完成確認
- 重大爭議事項已處理完成

### 必要產出

- 結果確認會議紀錄
- 修正後風險與問題清單
- 改善優先順序列表

---

## Milestone 6：結案資料備妥（Closure Ready）

### 對應流程階段

結案準備與交付階段

### 完成條件

- 專案所有執行產出文件已齊備
- 資料可供後續報告製作或改善行動使用

### 必要產出

- 專案執行紀錄彙整包
- 原始資料與分析資料集
- 會議與決策紀錄
- 結案確認文件

---

## Milestone 7：專案正式結案（Project Closed）

### 目的

在專案管理系統中正式關閉專案，完成歸檔。

### 完成條件

- 結案確認完成
- 專案狀態已標記為 Closed

### 必要產出

- 專案狀態更新紀錄
- 專案歷史資料封存位置說明

---

## 建議的工作板 Milestone 欄位結構

以下欄位可用於任何專案管理工具：

| Milestone                     | Status | Owner | Start Date | Due Date | Completion Criteria |
| ----------------------------- | ------ | ----- | ---------- | -------- | ------------------- |
| M0 Project Initiated          |        |       |            |          | 專案建立完成        |
| M1 Kickoff Completed          |        |       |            |          | 工作計畫書完成      |
| M2 Planning Completed         |        |       |            |          | 檢測計畫核准        |
| M3 Field Assessment Completed |        |       |            |          | 檢測項目全部完成    |
| M4 Initial Analysis Completed |        |       |            |          | 風險分級完成        |
| M5 Findings Validated         |        |       |            |          | 結果確認完成        |
| M6 Closure Ready              |        |       |            |          | 結案資料齊備        |
| M7 Project Closed             |        |       |            |          | 專案關閉            |

---

## 補充說明

- 本 Milestone List 可作為所有資安健診專案的共用工作板模板
- 專案進度以 Milestone 完成度為主，不以報告書產出為唯一依據
- 可依組織需求擴充角色、工具或合規欄位

本文件可直接納入公司專案管理標準作業流程（SOP）。
