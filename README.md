## 主要分類

### 關鍵問題點

- 選單邏輯：附加人員選單選擇後無法修改，需退出重進（議題 #1）。
- 查詢失效：客戶主/子分類清除查詢功能無效（議題 #4）。
- 儲存異常：產品規格無資料時無法儲存，或預設狀態邏輯錯誤（議題 #2, #6）。
- 提示缺失：專案代碼防呆提示未出現（議題 #3）。
- 顯示錯誤：專案收支、合約資訊等頁面在特定操作下顯示不完全或無法編輯。

### 功能新增（系統優化）摘要

#### 重點建議項目

- 文件管理強化：新增「需求書/規格書」上傳位元，並將「契約書」更名為「合約書/服務議定書」（議題 #102）。
- 流程自動化：業務報價單確認後，自動連動至專案收支（議題 #105）。
- 介面與彈性：增加支出選項下拉式選單（如：LA 成本、驗證費），減少手打錯誤並新增「其他」備註（議題 #105）。
- 權限與邏輯：調整 EIP 專案支出與旅費分類定義，使其更貼近財務報帳邏輯（議題 #103, #104）。
